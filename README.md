# Ghost Filter

- Scary Ghost Effect for Instagram 👻 built using Spark AR ✨
- Overlay's user's face 😃 with a Ghost Mask 🎭 
- Scary noise on mouth opening 😱 
- Continuous horror background music 😨 
- Live on Instagram and Facebook 😍
